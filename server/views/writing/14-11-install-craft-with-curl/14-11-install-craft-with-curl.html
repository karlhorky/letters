Sometimes you need to install something programmatically, such as when you're setting up a server from scratch (as you might if you were working with Digital Ocean). In those instances, you want to use wget or curl.

If you're installing Craft, you can get the zip file at http://buildwithcraft.com/latest.zip?accept_license=yes. A stack exchange post with more specifics can be found here.
